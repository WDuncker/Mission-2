Special Equipment & features

  • High-Mobility Aerial Tactics (HiMAT) Mode

    • The HiMAT system unfolds the eight wings of the mobile weapon platform mounted to the rear of the unit. This provides increased maneuverability during aerial combat. In space this system also enhances the Strike Freedom's AMBAC (Active Mass Balance Auto-Contro) capability.

  • Multi Lock-On System

    • This is a system used to target multiple enemy units and attack simultaneously. Effective use of the system requires a high level of spacial awareness from the pilot. The system can also be used when docked with the METEOR unit.

  • Full Burst Mode

    • When Full Burst Mode is engaged, the Strike Freedom uses all of it's rifles, DRAGOONS, railguns and it's multi-phase beam cannonat once. It is often used alongside the Multi Lock-On System. The DRAGOONS grant exceptional range and 360 degree firing capability. Unfortunately due to Earth's gravity, the DRAGOONS are not able to be used during in-atmosphere engagements.

  • "Voiture Lumiere" Propulsion System

    • The propulsion system is mounted on the unit's mobile weapon wing. It uses an internal laser and converts this energy into strong light pressure and uses this as thrust. It allows for incredible acceleration and enhances the unit's evasive abilities.

  • Hyper Deuterion Engine

    • The engine installed in the Strike Freedom is a hybrid system using a nuclear reactor alongside a deuterion beam energy tranfer system. This engine provides an incredibly high output and supplies enough power for the unit to use the METEOR system.

  • Neutron Jammer Canceller

    • This system disables the effects of N-Jammers (a series of nodes fired into the Earth's crust to prevent the development and use of nuclear technology). It allows the Strike Freedom the ability to operate on earth without causing it's engine to fail.

  • Operating System

    • The Strike Freedom boasts the Generation Unsubdued Nuclear Drive Assault Module (G.U.N.D.A.M) complex operating system. The system has unrivalled response times to pilot inputs and allows complete freedom of movement for the unit.

  • Variable Phase Shift Armour

    • The unit is equipped with Variable Phase Shift Armour, an updated version of the original Phase-Shift system used by the GAT-X105 Strike Gundam and it's counterpart suits developed alongside it. It allows for the sutomatic changing of the armour's strength, colour and power consumption during combat. These must be programmed as presets by the pilot before deployment.

  • METEOR Unit

    • The METEOR unit is a large mobile weapons platform that can be docked to the Strike Freedom. It is equipped with powerful thruster, battleship caliber beam cannons, large beam swords and an numerous array of heavy missile launchers.
